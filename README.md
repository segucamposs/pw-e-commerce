# SWAP Podcast — Website

University project for **ITBA 71.38 - Programación Web**.

A landing website for [SWAP Podcast](https://open.spotify.com/show/1t25iC8KdPXDZ9BUr1KgxY), a Spanish-language podcast for ambitious young people (16–25) in Argentina and LATAM. Hosted by Segu and Francisco Bottaro.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Main landing page — sections for guests to learn about SWAP and apply to participate |
| `guest.html` | QR code page — mobile-first overview of the podcast for potential guests |

## Tech Stack

Built progressively over 16 weeks:

- **Now:** Static HTML + CSS + JavaScript
- **E4:** React + Next.js
- **E5:** Supabase (database)
- **E6:** Mercado Pago (payments & webhooks)

## Deploy

Hosted on [Vercel](https://vercel.com). Auto-deploys on push to `main`. Preview deploys on every PR.
